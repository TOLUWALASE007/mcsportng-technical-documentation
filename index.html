Introduction

McSportng is a comprehensive sports management platform designed to support organizations, teams, coaches, players, agents, and administrators across multiple sports disciplines. The platform provides end-to-end solutions for team management, player development, statistics tracking, compliance management, and financial operations.

This technical documentation serves as a complete manual test guide and reference for understanding the application's architecture, features, and workflows.

Supported Sports

  • Football (Soccer) - Goals, assists, cards, formations, match events
  • Baseball - Batting, pitching, fielding stats with inning tracking
  • Golf - Scorecards, handicaps, tee times, kitty management
  • Other Sports - Extensible framework for additional sports

Target Audience

  • QA Engineers - Manual testing procedures and test case references
  • Developers - Technical implementation details and API endpoints
  • Product Owners & Stakeholders - Feature overview and business logic

Platform Overview

The McSportng platform is organized into role-based dashboards, each tailored to specific user types and their responsibilities:

Organization Admin

Manage teams, leagues, compliance, billing, and organization-wide settings

Team Admin

Roster management, match statistics, player performance tracking

Coach

Training schedules, lesson booking, stats verification, performance analysis

Player/Athlete

Profile management, media uploads, career history, achievements

Super Admin

Platform-wide control, organization onboarding, user management, financials

Public Access

Marketing pages, public statistics, contact forms, pricing information
