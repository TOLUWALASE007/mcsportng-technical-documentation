Coach Dashboard

User Role: Coach

Base Path: /coach

Training Schedule

Path: /coach/training-schedule

  • Create training sessions
  • Set date, time, and location
  • Assign players to sessions
  • Track attendance
  • Add training notes and objectives

Lesson Booking

Path: /coach/lesson-booking

  • Set availability for private lessons
  • View booking requests
  • Approve or decline lesson bookings
  • Manage lesson pricing
  • Track completed lessons

Stats Verification

Path: /coach/stats-verification

  • Review player statistics
  • Verify accuracy of submitted stats
  • Add coaching notes to player performance
  • Flag discrepancies for review

League Performance

Path: /coach/league-performance

  • View team standings
  • Analyze player performance trends
  • Compare stats across seasons
  • Export performance reports
